# Python Contact Book 

Description: Python Contact Book is a practice project which covers basic python 
programming and shows modular programming using functions 

## Features
        
          - Add new contacts
          - Show all saved contacts
          - Search any contact which is saved 
          - Delete saved contacts

## Concepts Used

          - For Loop 
          - Functions
          - Dictionary 
          - Control Structure (if/else)
